Below are the instructions to get started with the Iconic theme.

Setup

Install Blockify parent theme:
  • In your site's admin panel, go to Appearance > Themes and click Add New.
  • Type Blockify in the search field.
  • Click Install. You are now ready to install the Iconic child theme!
Install Iconic child theme:
  • In your BlockifyWP.com account, navigate to Downloads.
  • Download the Iconic.zip file.
  • In your site's admin panel, go to Appearance > Themes and click Add New.
  • Upload the Iconic.zip file
  • Click Install and then Activate to start using the theme.
  • Navigate to Appearance > Editor.
  • You are now ready to begin customizing your theme ðŸŽ‰
